# Cloud Resume Challenge - Azure Edition

## Introduction
This repository contains my submission for the Cloud Resume Challenge, a project designed to showcase cloud skills, specifically using Microsoft Azure. The challenge involves building a cloud-based resume with various technical components.

## Project Overview
The Cloud Resume Challenge requires the participant to build a resume as a static website, host it on Azure, and incorporate various cloud services and features. The goal is to demonstrate proficiency in cloud concepts and practical Azure skills.

### Key Features
- **Static Website Hosting**: The resume is a static HTML/CSS website hosted on Azure Blob Storage with Azure CDN for global distribution.
- **Database Integration**: Visitor count is tracked using Azure Functions and stored in an Azure Cosmos DB.
- **API Implementation**: An Azure Function is implemented to create an API for updating and retrieving the visitor count from the database.
- **CI/CD Pipeline**: Automated deployment using GitHub Actions for continuous integration and delivery.

## Technologies Used

- Azure Blob Storage
- Azure CDN
- Azure Functions
- Azure Cosmos DB
- GitHub Actions
